This project presents a two-page interactive Power BI dashboard designed to analyze Near-Earth Asteroids (NEOs) using scientifically reliable orbital and physical data.
The dashboard emphasizes risk assessment, orbital behavior, proximity to Earth, and data reliability, going beyond simple visualization to provide structured analytical insights.

ğŸ¯ Objectives of the Project

The dashboard was developed to achieve the following objectives:

Identify Near-Earth Objects (NEOs) and Potentially Hazardous Asteroids (PHAs)

Assess Earth proximity risk using Minimum Orbit Intersection Distance (MOID)

Analyze asteroid size and impact potential

Detect large but dim (hard-to-observe) asteroids

Study orbital characteristics such as eccentricity and inclination

Evaluate data reliability using RMS orbit uncertainty

Prioritize asteroids for scientific observation and monitoring

Enable object-level drill-down analysis for transparency and validation
